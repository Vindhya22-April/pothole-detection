# 🚧 Road Pothole Detection using CNN

## 📌 Project Description
This project focuses on detecting potholes on roads using Convolutional Neural Networks (CNN). It helps in identifying damaged road areas from images to improve road safety and maintenance.

---

## 🚀 Features
- Detects potholes from road images
- Uses CNN model for accurate classification
- Image preprocessing and model training
- Prediction module for real-time detection

---

## 🛠️ Technologies Used
- Python
- OpenCV
- TensorFlow / Keras
- NumPy
- Matplotlib

---

## 📂 Project Structure
- app.py → Main application file  
- model.py → CNN model definition  
- train.py → Model training  
- predict.py → Prediction logic  
- modules.py → Helper functions  

## 📊 Dataset

The dataset used for this project is from Kaggle:
Annotated Potholes with Severity Levels

- Contains labeled images of potholes
- Includes severity classification (low, medium, high)
- Used for training and testing the CNN model

Dataset Link:
https://www.kaggle.com/datasets/idanbaru/annotated-potholes-with-severity-levels


## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   python app.py
